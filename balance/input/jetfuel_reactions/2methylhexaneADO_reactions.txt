CO2 + H2O → HCO3- + Hplus
ATP + Acetyl-CoA + HCO3- → ADP + Orthophosphate + Malonyl-CoA
Acetyl-CoA + Malonyl-CoA → Acetoacetyl-CoA + CO2 + CoA
Acetoacetyl-CoA + NADPH +Hplus → 3R-3-Hydroxybutanoyl-CoA + NADPplus
3R-3-Hydroxybutanoyl-CoA → But-2-enoyl-CoA + H2O
But-2-enoyl-CoA + NADPH + Hplus → Butanoyl-CoA + NADPplus
Butanoyl-CoA → 2-methyl-Propanoyl-CoA
Malonyl-CoA + 2-methyl-Propanoyl-CoA → CoA + CO2 + 2-methyl-Oxopentanoyl-CoA
2-methyl-Oxopentanoyl-CoA + NADH + Hplus → S-2-methyl-Hydroxypentanoyl-CoA + NADplus
S-2-methyl-Hydroxypentanoyl-CoA → 2-methyl-trans-pent-2-enoyl-CoA + H2O
2-methyl-trans-pent-2-enoyl-CoA + NADPH + Hplus → 2-methyl-Pentanoyl-CoA + NADPplus
Malonyl-CoA + 2-methyl-Pentanoyl-CoA → CoA + CO2 + 2-methyl-Oxoheptanoyl-CoA
2-methyl-Oxoheptanoyl-CoA + NADH + Hplus → S-2-methyl-Hydroxyheptanoyl-CoA + NADplus
S-2-methyl-Hydroxyheptanoyl-CoA → 2-methyl-trans-hept-2-enoyl-CoA + H2O
2-methyl-trans-hept-2-enoyl-CoA + NADPH + Hplus → 2-methyl-Heptanoyl-CoA + NADPplus
2-methyl-Heptanoyl-CoA + NADH + Hplus → CoA + 2-methyl-Heptanal + NADplus
2-methyl-Heptanal + Oxygen + 2*NADPH + 2*Hplus → 2-methyl-Hexane + Formate + H2O + 2*NADPplus
Formate → CO2 + NADH + Hplus
NADH + NADPplus → NADPH + NADplus
