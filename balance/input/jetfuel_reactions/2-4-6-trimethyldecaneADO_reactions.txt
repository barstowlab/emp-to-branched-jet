ATP + Propionyl-CoA + HCO3- → ADP + Orthophosphate + (S)-Methylmalonyl-CoA
Acetyl-CoA + Malonyl-CoA → Acetoacetyl-CoA + CO2 + CoA
Acetoacetyl-CoA + NADPH + Hplus → 3R-3-Hydroxybutanoyl-CoA + NADPplus
3R-3-Hydroxybutanoyl-CoA → But-2-enoyl-CoA + H2O
But-2-enoyl-CoA + NADPH + Hplus → Butanoyl-CoA + NADPplus
Butanoyl-CoA → 2-methyl-Propanoyl-CoA
(S)-Methylmalonyl-CoA + 2-methyl-Propanoyl-CoA → CoA + CO2 + 2-4-dimethyl-Oxopentanoyl-CoA
2-4-dimethyl-Oxopentanoyl-CoA + NADH + Hplus → S-2-4-dimethyl-Hydroxypentanoyl-CoA + NADplus
S-2-4-dimethyl-Hydroxypentanoyl-CoA → 2-4-dimethyl-trans-pent-2-enoyl-CoA + H2O
2-4-dimethyl-trans-pent-2-enoyl-CoA + NADPH + Hplus → 2-4-dimethyl-Pentanoyl-CoA + NADPplus
(S)-Methylmalonyl-CoA + 2-4-dimethyl-Pentanoyl-CoA → CoA + CO2 + 2-4-6-trimethyl-Oxoheptanoyl-CoA
2-4-6-trimethyl-Oxoheptanoyl-CoA + NADH + Hplus → S-2-4-6-trimethyl-Hydroxyheptanoyl-CoA + NADplus
S-2-4-6-trimethyl-Hydroxyheptanoyl-CoA → 2-4-6-trimethyl-trans-hept-2-enoyl-CoA + H2O
2-4-6-trimethyl-trans-hept-2-enoyl-CoA + NADPH + Hplus → 2-4-6-trimethyl-Heptanoyl-CoA + NADPplus
Malonyl-CoA + 2-4-6-trimethyl-Heptanoyl-CoA → CoA + CO2 + 2-4-6-trimethyl-Oxononanoyl-CoA
2-4-6-trimethyl-Oxononanoyl-CoA + NADH + Hplus → S-2-4-6-trimethyl-Hydroxynonanoyl-CoA + NADplus
S-2-4-6-trimethyl-Hydroxynonanoyl-CoA → 2-4-6-trimethyl-trans-non-2-enoyl-CoA + H2O
2-4-6-trimethyl-trans-non-2-enoyl-CoA + NADPH + Hplus → 2-4-6-trimethyl-Nonanoyl-CoA + NADPplus
Malonyl-CoA + 2-4-6-trimethyl-Nonanoyl-CoA → CoA + CO2 + 2-4-6-trimethyl-Oxoundecanoyl-CoA
2-4-6-trimethyl-Oxoundecanoyl-CoA + NADH + Hplus → S-2-4-6-trimethyl-Hydroxyundecanoyl-CoA + NADplus
S-2-4-6-trimethyl-Hydroxyundecanoyl-CoA → 2-4-6-trimethyl-trans-undec-2-enoyl-CoA + H2O
2-4-6-trimethyl-trans-undec-2-enoyl-CoA + NADPH + Hplus → 2-4-6-trimethyl-Undecanoyl-CoA + NADPplus
2-4-6-trimethyl-Undecanoyl-CoA + NADH + Hplus → CoA + 2-4-6-trimethyl-Undecanal + NADplus
2-4-6-trimethyl-Undecanal + Oxygen + 2*NADPH + 2*Hplus → 2-4-6-trimethyl-Decane + Formate + H2O + 2*NADPplus
Formate → CO2 + NADH + Hplus
NADH + NADPplus → NADPH + NADplus
