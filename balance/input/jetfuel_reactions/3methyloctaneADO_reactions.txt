CO2 + H2O → HCO3- + Hplus
ATP + Acetyl-CoA + HCO3- → ADP + Orthophosphate + Malonyl-CoA
ATP + Propionyl-CoA + HCO3- → ADP + Orthophosphate + (S)-Methylmalonyl-CoA
Propionyl-CoA + (S)-Methylmalonyl-CoA → 5-4-methyl-Acetoacetyl-CoA + CO2 + CoA
5-4-methyl-Acetoacetyl-CoA + NADPH + Hplus → 3R-4-methyl-3-Hydroxypentanoyl-CoA + NADPplus
3R-4-methyl-3-Hydroxypentanoyl-CoA → 4-methyl-Pent-2-enoyl-CoA + H2O
4-methyl-Pent-2-enoyl-CoA + NADPH + Hplus → 4-methyl-Pentanoyl-CoA + NADPplus
Malonyl-CoA + 4-methyl-Pentanoyl-CoA → CoA + CO2 + 4-methyl-Oxoheptanoyl-CoA
4-methyl-Oxoheptanoyl-CoA + NADH + Hplus → S-4-methyl-Hydroxyheptanoyl-CoA + NADplus
S-4-methyl-Hydroxyheptanoyl-CoA → 4-methyl-trans-hept-2-enoyl-CoA + H2O
4-methyl-trans-hept-2-enoyl-CoA + NADPH + Hplus → 4-methyl-Heptanoyl-CoA + NADPplus
Malonyl-CoA + 4-methyl-Heptanoyl-CoA → CoA + CO2 + 4-methyl-Oxononanoyl-CoA
4-methyl-Oxononanoyl-CoA + NADH + Hplus → S-4-methyl-Hydroxynonanoyl-CoA + NADplus
S-4-methyl-Hydroxynonanoyl-CoA → 4-methyl-trans-non-2-enoyl-CoA + H2O
4-methyl-trans-non-2-enoyl-CoA + NADPH + Hplus → 4-methyl-Nonanoyl-CoA + NADPplus
4-methyl-Nonanoyl-CoA + NADH + Hplus → CoA + 4-methyl-Nonanal + NADplus
4-methyl-Nonanal + Oxygen + 2*NADPH + 2*Hplus → 3-methyl-Octane + Formate + H2O + 2*NADPplus
Formate → CO2 + NADH + Hplus
NADH + NADPplus → NADPH + NADplus
