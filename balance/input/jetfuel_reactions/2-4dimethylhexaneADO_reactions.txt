Propionyl-CoA + CO2 → (S)-Methylmalonyl-CoA
CO2 + H2O → HCO3- + Hplus
ATP + Acetyl-CoA + HCO3- → ADP + Orthophosphate + Malonyl-CoA
Acetyl-CoA + Malonyl-CoA → Acetoacetyl-CoA + CO2 + CoA
Acetoacetyl-CoA + NADPH + Hplus → 3R-3-Hydroxybutanoyl-CoA + NADPplus
3R-3-Hydroxybutanoyl-CoA → But-2-enoyl-CoA + H2O
But-2-enoyl-CoA + NADPH + Hplus → Butanoyl-CoA + NADPplus
Butanoyl-CoA → 2-methyl-Propanoyl-CoA
(S)-Methylmalonyl-CoA + 2-methyl-Propanoyl-CoA → CoA + CO2 + 2-4-dimethyl-Oxopentanoyl-CoA
2-4-dimethyl-Oxopentanoyl-CoA + NADH + Hplus → S-2-4-dimethyl-Hydroxypentanoyl-CoA + NADplus
S-2-4-dimethyl-Hydroxypentanoyl-CoA → 2-4-dimethyl-trans-pent-2-enoyl-CoA + H2O
2-4-dimethyl-trans-pent-2-enoyl-CoA + NADPH + Hplus → 2-4-dimethyl-Pentanoyl-CoA + NADPplus
Malonyl-CoA + 2-4-dimethyl-Pentanoyl-CoA → CoA + CO2 + 2-4-dimethyl-Oxoheptanoyl-CoA
2-4-dimethyl-Oxoheptanoyl-CoA + NADH + Hplus → S-2-4-dimethyl-Hydroxyheptanoyl-CoA + NADplus
S-2-4-dimethyl-Hydroxyheptanoyl-CoA → 2-4-dimethyl-trans-hept-2-enoyl-CoA + H2O
2-4-dimethyl-trans-hept-2-enoyl-CoA + NADPH + Hplus → 2-4-dimethyl-Heptanoyl-CoA + NADPplus
2-4-dimethyl-Heptanoyl-CoA + NADH + Hplus → CoA + 2-4-dimethyl-Heptanal + NADplus
2-4-dimethyl-Heptanal + Oxygen + 2*NADPH + 2*Hplus → 2-4-dimethyl-Hexane + Formate + H2O + 2*NADPplus
Formate → CO2 + NADH + Hplus
NADH + NADPplus → NADPH + NADplus
