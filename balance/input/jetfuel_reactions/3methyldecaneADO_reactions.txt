CO2 + H2O → HCO3- + Hplus
ATP + Propionyl-CoA + HCO3- → ADP + Orthophosphate + (S)-Methylmalonyl-CoA
ATP + Acetyl-CoA + HCO3- → ADP + Orthophosphate + Malonyl-CoA
Propionyl-CoA + Malonyl-CoA → 5-Acetoacetyl-CoA + CO2 + CoA
5-Acetoacetyl-CoA + NADPH + Hplus → 3R-3-Hydroxypentanoyl-CoA + NADPplus
3R-3-Hydroxypentanoyl-CoA → Pent-2-enoyl-CoA + H2O
Pent-2-enoyl-CoA + NADPH + Hplus → Pentanoyl-CoA + NADPplus
(S)-Methylmalonyl-CoA + Pentanoyl-CoA → CoA + CO2 + 3-Oxoheptanoyl-CoA
3-Oxoheptanoyl-CoA + NADH + Hplus → S-Hydroxyheptanoyl-CoA + NADplus
S-Hydroxyheptanoyl-CoA → trans-hept-2-enoyl-CoA + H2O
trans-hept-2-enoyl-CoA + NADPH + Hplus → Heptanoyl-CoA + NADPplus
(S)-Methylmalonyl-CoA + Heptanoyl-CoA → CoA + CO2 + 8-methyl-3-Oxononanoyl-CoA
8-methyl-3-Oxononanoyl-CoA + NADH + Hplus → 8-methyl-S-Hydroxynonanoyl-CoA + NADplus
8-methyl-S-Hydroxynonanoyl-CoA → 8-methyl-trans-non-2-enoyl-CoA + H2O
8-methyl-trans-non-2-enoyl-CoA + NADPH + Hplus → 8-methyl-Nonanoyl-CoA + NADPplus
Malonyl-CoA + 8-methyl-Nonanoyl-CoA → CoA + CO2 + 8-methyl-3-Oxoundecanoyl-CoA
8-methyl-3-Oxoundecanoyl-CoA + NADH + Hplus → 8-methyl-S-Hydroxyundecanoyl-CoA + NADplus
8-methyl-S-Hydroxyundecanoyl-CoA → 8-methyl-trans-undec-2-enoyl-CoA + H2O
8-methyl-trans-undec-2-enoyl-CoA + NADPH + Hplus → 8-methyl-Undecanoyl-CoA + NADPplus
8-methyl-Undecanoyl-CoA + NADH + Hplus → CoA + 8-methyl-Undecanal + NADplus
8-methyl-Undecanal + Oxygen + 2*NADPH + 2*Hplus → 3-methyl-Decane + Formate + H2O + 2*NADPplus
Formate → CO2 + NADH + Hplus
NADH + NADPplus → NADPH + NADplus
